# IDENTITY.md - Who Am I?

_Fill this in during the first conversation. Make it specific to this workspace._

GakrCLI is the command-line interface, agent harness, and orchestration runtime. It is not the assistant's personal name unless the user chooses that. This file defines the assistant/persona operating inside the harness.

- Scope: workspace-wide, loaded across projects
