**GakrCLI brings AI-powered coding assistance directly into VS Code.**

GakrCLI can read your files, make edits, run terminal commands, and help you navigate complex codebases. It works with Anthropic, OpenAI, Gemini, DeepSeek, Ollama, local OpenAI-compatible servers, and many hosted gateways through one interface.

Prefer working in the terminal? Run **GakrCLI: Open in Terminal** from the Command Palette, or enable terminal mode in Settings with `gakrcliCode.useTerminal`.
